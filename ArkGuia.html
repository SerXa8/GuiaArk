1. Primeros pasos estratégicos

  • Prioriza el desbloqueo de Engramas Tech y Kibble: Esenciales para acelerar progreso y tameos.
  • Ubicación inicial: Cerca de agua, metal ligero, y madera; evita zonas PvP densas.
  • Recolecta rápido: Usa Castoroides y Doedicurus lo antes posible para madera y piedra.
  • Taming eficiente: Rifles tranquilizantes y Kibble para minimizar riesgo y acelerar tameo.

3.1 Mapa y zonas clave para farmear y bases

Conoce las mejores ubicaciones del mapa Fjordur para recolectar recursos esenciales y establecer bases estratégicas que te den ventaja en PvP y PvE.

Zonas para farmear recursos

  • Metal: Montañas al noreste, volcanes cercanos a Vanaheim. Ideal para farmear con Magmasaur o T-Rex.
  • Piedra: Ríos y colinas en Midgard, zonas cercanas a los lagos centrales con Doedicurus.
  • Madera: Bosques densos de Redwood en el suroeste y norte, con Castoroides o Therizinosaur para farmear rápido.
  • Oil: Fondos marinos en las zonas congeladas y costas del sur. Usa dinos acuáticos fuertes como Megalodon o Tusoteuthis.
  • Elementos raros: Cuevas profundas y volcanes activos para conseguir cristales y rarezas con Glowtail y Gacha.

Zonas recomendadas para bases

  • Plataformas Redwood: Excelente para bases arbóreas con buena visibilidad y defensa natural.
  • Montañas aisladas: Sitios difíciles de acceder para enemigos, perfectos para bases Tech defensivas.
  • Cuevas protegidas: Bases ocultas ideales para almacenar loot valioso y proteger recursos clave.
  • Islas apartadas: Perfectas para criaderos, almacenamiento seguro y reducción de ataques PvP.
  • Áreas cercanas a recursos: Siempre prioriza ubicaciones que te permitan farmear rápido sin grandes desplazamientos.

4. Construcción avanzada de bases

  • Base Tech: Muros dobles, techos dobles, torretas en todas las entradas y puntos vulnerables.
  • Defensa escalonada: Spike walls, plantas X, torretas automáticas, y dinos guardianes.
  • Ubicación estratégica: Bases elevadas o dentro de cuevas para ventaja táctica.
  • Almacenamiento dividido: Divide loot clave en cofres separados para minimizar pérdidas.
  • Uso intensivo de Cryopods: Mantén dinos valiosos para raids y defensa listos para desplegar.

5. Estrategias PvP avanzadas

  • Raideos nocturnos: Usa Ghillie, Shadowmane y dinos veloces para ataques sorpresa.
  • Distracciones y señuelos: Dinos pequeños para distraer torretas y limpiar trampas.
  • Control del mapa: Domina puntos clave para asegurar recursos y bases enemigas.
  • Coordinación en equipo: Roles claros: scout, tanque, sniper y healer.
  • Runas y artefactos: Usa buffs para raid y defensa, maximizando tus chances.

6. Tameo rápido y eficiente

  • Rifles tranquilizantes con dardos narcóticos o tranqulizantes para minimizar riesgos.
  • Kibble especializado para cada especie para acelerar el tameo.
  • Trampas de piedra para tameos seguros de dinos agresivos (Argentavis, Rex).
  • Dinos protectores como Therizinosaur durante tameos difíciles.

7. Farming optimizado

  • Metal: Magmasaur con set minería para producción máxima.
  • Madera: Castoroides y Therizinosaur para grandes cantidades.
  • Piedra: Doedicurus es el mejor recolector.
  • Oil: Dinos acuáticos como Tusoteuthis o Megalodon para farmear seguro.
  • Recursos raros: Usa gachas personalizadas en zonas seguras para loot extra.

8. Exploración y uso de cuevas

  • Prepárate con armas, armadura y alimentos adecuados antes de entrar.
  • Usa dinos resistentes y con buena movilidad para explorar rápidamente.
  • Glowtail es esencial para iluminación natural dentro de las cuevas.
  • Evita cuevas muy profundas si no tienes experiencia o dinos fuertes.
  • Las cuevas esconden loot valioso y artefactos para desbloquear Engramas.

9. Consejos extras para dominar Fjordur

  • Comunícate constantemente con tu equipo para evitar sorpresas PvP.
  • Recuerda usar las herramientas de construcción rápida para ahorrar tiempo.
  • Prioriza proteger tus dinos con Cryopods durante raids enemigos.
  • Realiza scouting frecuente para anticipar movimientos enemigos.
  • Optimiza el peso cargado en dinos para viajes más eficientes.

10. Crianza y mutaciones de estadísticas

  • ¿Por qué criar dinos? Para mejorar estadísticas base, obtener mejores genes y crear dinos más fuertes para combate, farm o PvP.
  • Proceso de crianza:
    • Empareja dos dinos compatibles (misma especie y género opuesto).
    • Coloca una incubadora cerca con huevos fertilizados para que eclosionen.
    • Cuida y alimenta al bebé hasta que crezca; durante este proceso, heredará estadísticas de sus padres.
  • Mutaciones de estadísticas:
    • Cada vez que nace un bebé, tiene una pequeña probabilidad de mutar una o más estadísticas, incrementándolas.
    • Las mutaciones se reflejan en barras adicionales en las estadísticas del dino (verás los "mut" o +1, +2, etc.).
    • Las mutaciones aumentan la salud, daño, resistencia, etc., haciendo al dino más poderoso.
    • Para maximizar mutaciones, cría dinos con estadísticas ya altas y usa un sistema de crianza para seleccionar mutaciones acumulativas.
  • Consejos para crianza eficiente:
    • Usa dinos con buenas estadísticas base para empezar.
    • Cría consistentemente para aumentar las mutaciones con cada generación.
    • Almacena los mejores dinos y desecha los que no tengan mutaciones interesantes.
    • Considera usar mods o servidores que mejoren el sistema de crianza si quieres acelerar el proceso.
Tip: Llevar un registro de las mutaciones y usar nombres para identificar líneas de crianza ayuda a mantener control y mejorar más rápido.
